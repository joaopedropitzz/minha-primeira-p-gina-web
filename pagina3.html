O CSS documento de estilização em cascata criado em 1993, com sua primeira versão amplamente utilizada em 1997. A linguagem CSS permite definir regras que definem características para os elementos HTML como cor, borda e comportamentos, além de animações e transições.

Ao longo dos anos o css evoluiu e hoje temos o CSS 1, 2 e 3, cada uma das versões adicionou novos recursos, que eram demandas no período.

  1. CSS 1 1997 - propriedades básicas de estilização e formatação, além de controle do posicionamento dos elementos, html com display, propriedades float e position
  2. CSS 2 1998 - adição de propriedades de animação e transição e melhoria do position
  3. CSS 3 2008 - incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento como display:flex (2008) e grid (2013)

Uma regra css pode ser definida com a seguinte sintaxe

            Seletor{
                Propriedade1:Valor;
                Propriedade2:Valor;

                PropriedadeN:Valor;
            }

Podemos incluir código CSS de 3 formas diferentes

CSs externo
Quando definirmos as regras CSS a serem utilizadas por meio de um arquivo externo .css, vinculando ele ao html por meio da tag < link > no head do documento html
CSS incorporado "no documento"
Quando definimos as regras CSS por meio da tag <style >
CSS inline
Quando definimos código css de estilização no próprio elemento HTML como: Texto de cor vermelha com CSS inline
